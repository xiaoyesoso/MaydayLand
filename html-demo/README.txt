MaydayLand · 五月天·城市漫游
=====================================

交互式可体验 HTML 演示 · 比赛评审交付物

【交付物说明】
本压缩包包含 MaydayLand（五月天·城市漫游）的交互式 HTML 演示，是比赛评审的可点击交付物。

【文件清单】
- index.html  : 主页面（含全部 CSS 样式与页面结构）
- app.js      : 交互逻辑（含种子数据、路由、Canvas 卡片合成、localStorage 等）
- README.txt  : 本说明文件

【使用方法】
1. 解压本压缩包到任意目录
2. 双击 index.html 用浏览器打开（推荐 Chrome / Safari / Edge 最新版）
3. 无需安装任何依赖，无需启动服务器，零配置即开即体验
4. 首次进入会有 3 步引导，点击"开始漫游"即可体验核心玩法

【核心功能】
1. 发现页（歌词角落地图）
   - 五球配色 LyricBanner 横幅 + 启动页动画
   - 11 个情绪标签筛选（温柔/倔强/同款/地标/限定/首展/官方等）
   - 歌词关键词搜索
   - 12 个真实五迷角落卡片列表（2026 最新数据）
   - SVG 自绘城市地图 + 角落标记点
   - 右下角悬浮按钮"推荐新角落"，可提交 UGC 角落（localStorage 持久化）

2. 角落详情页
   - 真实地点配图 + 匹配歌词 + 五迷推荐语
   - 五迷暗号区块（对暗号享隐藏权益）
   - "我要打卡" / "我要对暗号" 双按钮
   - 轻量评论区（localStorage 持久化）

3. 打卡卡片页
   - 上传/拍摄现场照片（<input type="file">）
   - Canvas 合成 3 套模板：复古拍立得 / 演唱会票根 / 极简胶片
   - 无照片时自动生成歌词水印装饰
   - 一键保存卡片 PNG / 分享给好友（Web Share API + 复制文案兜底）
   - 保存/分享成功触发五球 confetti 庆祝动画

4. 五迷暗号核销
   - 模拟一次性 token 二维码（Canvas 绘制）
   - 10 分钟倒计时
   - "模拟商家核销"按钮 → 解锁主题歌徽章
   - 首次解锁触发庆祝动画

5. 演唱会页
   - 鸟巢 55 场收官 banner
   - 4 站巡演日程（北京/台北/长沙/上海）
   - 5 条官方动态（资讯/票务/活动）

6. 我的页
   - 打卡足迹（按时间倒序）
   - 五迷徽章墙（8 枚主题歌徽章，对暗号解锁）
   - 分享统计
   - 我的推荐（用户提交的 UGC 角落审核状态）

【视觉系统】
- 五球配色：蓝 #3B7DD8 / 粉 #FF6B9D / 黄 #FFD23F / 绿 #4ECDC4 / 紫 #9B7EDE
- 卜卜元素：圆润几何 + 纯真兔耳朵 SVG 图标
- STAYREAL 风：粗黑标题 + 潮玩贴纸感标签 + 大圆角卡片
- 启动页：五球弹跳动画 + MAYDAYLAND 标题 + 副标题淡入
- 动效：页面 slide-in 转场、模板切换高亮、暗号弹窗 popIn、保存/分享 confetti

【技术实现】
- 单文件自包含（index.html + app.js）
- 手机外壳模拟（390×844，iPhone 比例）
- hash 路由页面栈（模拟小程序 navigateTo）
- Canvas 2D API 合成打卡卡片
- localStorage 管理评论/足迹/徽章/推荐
- Web Share API 分享 + 剪贴板兜底
- 浏览器 geolocation 定位体验
- 响应式设计（移动端自适应）
